Администратор
1) Создать клиента (затребовать имя, фамилию, почту, пароль)
2) Изменить клиента (затребовать код клиента, можно изменять только почту, пароль)
3) Удалить клиента (затребовать код клиента, кроме клиента удалются все счета и карты этого клиента - каскадное удаление)
4) Просмотреть список клиентов 
4.1) Найти клиента по номеру
4.2) Найти клиента по почте
4.3) Найти клиентов по фамилии
5) Изменение конфигурации (процент, ограничение транзакций, штраф) затребовать название показателя и новое значение
6) Создать счет (затребовать код клиента, стартовый баланс (сколько денег на счет положить))
7) Удалить счет (затребовать номер счета, удалить все связанные карты - каскадное удаление)
8) Показать список счетов
9) Создать карту (затребовать номер счета, которому привязыается карта, номер карты, срок окончания действия карты, пин-код карты, csv-код (трехзначный код на обратной стороне карты))
10) Удалить карту (затребовать номер карты)
11) Показать список карт (затребовать номер счета либо если пусто показать все карты по всем счетам)
Операционист:
1) Пополнить баланс (затребовать тип (счет или карта), затребовать номер счета (карты), увеличить количество транзакций по счету, добавить тразакцию)
2) Снять деньги со счета (затребовать тип (счет или карта), затребовать номер счета (карты), увеличить количество транзакций по счету, добавить транзакцию)
3) Показать список счетов клиента (затребовать код клиента, код клиента-идентификационный номер по паспорту)
4) Просмотр баланса (затребовать тип (счет или карта), затребовать номер счета (карты), списать деньги за просмотр баланса, размер комиссии из конфига)
